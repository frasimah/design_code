# Vandersanden Scraper

Проект для скраппинга и обработки данных с сайта [Vandersanden](https://www.vandersanden.com).

## Структура проекта

```
lick_brick/
├── src/
│   ├── scraper/          # Модули для скраппинга
│   │   ├── __init__.py
│   │   ├── base.py       # Базовый класс скрапера
│   │   ├── products.py   # Скраппер продуктов
│   │   └── utils.py      # Вспомогательные функции
│   ├── processors/       # Модули для обработки данных
│   │   ├── __init__.py
│   │   └── cleaner.py    # Очистка и нормализация данных
│   └── models/           # Модели данных
│       ├── __init__.py
│       └── product.py    # Модель продукта
├── data/                 # Сохранённые данные
│   ├── raw/              # Сырые данные
│   └── processed/        # Обработанные данные
├── config/               # Конфигурация
│   └── settings.py
├── tests/                # Тесты
├── requirements.txt      # Зависимости
└── main.py               # Точка входа
```

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Скраппинг всех продуктов
python main.py scrape --all

# Скраппинг по категории
python main.py scrape --category "фасадный-кирпичи"

# Экспорт в JSON/CSV
python main.py export --format json --output data/products.json
```

## Данные

Для каждого продукта собираются:
- Название и артикул
- Цвет (базовый, дополнительные, нюансы)
- Текстура и способ производства
- Форматы и размеры
- Технические характеристики (вес, штук/м², штук в палете)
- Ссылки на проекты
- Изображения и BIM-модели

## Конфигурация AI

> [!IMPORTANT]
> **Модель Gemini: `gemini-3-flash-preview`**
> 
> Эта модель обязательна для парсинга PDF документов. НЕ менять на другие версии!
> Модель используется для:
> - Извлечения данных из PDF (DoP, CE сертификаты)
> - Перевода технической документации на русский язык
